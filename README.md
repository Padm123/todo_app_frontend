##  Tech Stack
- React (Vite) + Bootstrap
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
- Deployed via Vercel & Render

##  Features
- Signup and Login with email/password
- Authenticated dashboard
- Add, edit, and delete to-do items
- Each user sees only their tasks
- Protected routes with JWT
